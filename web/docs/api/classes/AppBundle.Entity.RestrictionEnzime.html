__constructor(\AppBundle\Entity\type  $args)

Constructor method for the RestEn class. It creates a new RestEn object and initializes its properties accordingly.

RestEn() behavior: If passed with make = 'custom', object will be added to aRestEnzimDB. If not, the function will attemp to retrieve data from aRestEnzimDB. If unsuccessful in retrieving data, it will return an error flag.
