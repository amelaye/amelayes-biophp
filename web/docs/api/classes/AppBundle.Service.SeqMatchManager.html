xlevdist()

xlevdist(\AppBundle\Service\type  $s, \AppBundle\Service\type  $t) : \AppBundle\Service\type

xlevdist() is an extended version of levdist() which accepts strings with length greater than 255 but not to exceed 1024 (which takes my CPU 18 seconds to compute).

The only drawback to xlevdist is that the cost of insertion, deletion, and replacement is fixed to 1. I have yet to find a way to allow custom values for these.

match()

match(\AppBundle\Service\type  $str1, \AppBundle\Service\type  $str2, \AppBundle\Service\type  $matrix, \AppBundle\Service\type  $equal, \AppBundle\Service\type  $partial = "+", \AppBundle\Service\type  $nomatch = ".") : string

The match() method accepts two sequence strings (not objects) of equal length, and returns a sequence match result string, according to the following rules: If there is an exact match, return the amino acid symbol.

If there is a partial match, return a plus sign. If there is no match, return a whitespace character.
